# Automation Architect Toolkit

A modern automation engineering platform built incrementally to demonstrate
enterprise-grade test architecture and current industry best practices.

The project will evolve over time to include:

- TypeScript
- Node.js
- Playwright
- API Testing
- Docker
- GitHub Actions
- Jenkins
- Performance Testing
- Mobile Automation
- Reporting
- Cloud Integrations
- Utilities
- Reference Architectures

Each technology is added as part of a structured learning roadmap with
production-quality implementations and documentation.
